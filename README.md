#### Source Code ####
The source code can be found in the src/ folder. 
To run, it requires:
- java 25 (https://www.oracle.com/java/technologies/downloads/) 
- the gson library (https://search.maven.org/artifact/com.google.code.gson/gson/2.13.1/jar). 
- In addition, the testing suite requires JUnit5 (https://central.sonatype.com/artifact/org.junit.jupiter/junit-jupiter-engine/5.14.1)

#### Running the code ####
The entry point for the program is: "main/Main.java". The program takes in the location of a config file and uses the file located at "configs/example_config.json" if none is provided.
